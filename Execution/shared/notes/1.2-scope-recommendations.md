## Statistical pipeline code (before Phase 2 production July 1)

R6 flagged that IP §12 specifies all statistical frameworks in prose but
**none are coded yet**. These must exist and be tested on pilot data before
Phase 2 production begins.

### Alpha-M (IP §12.1)
3. **Friedman + Nemenyi post-hoc test** — R or Python implementation. Tested on Phase 1 pilot MLFF data.
4. **ICC(2,k) and ICC(2,1) estimation with convergence-correction factor** — IP §1 + §10.3 note ~15-25% attenuation needs quantification. Need: code + validation + documented correction factor.
5. **Hierarchical bootstrap** — IP lists but resampling strategy (protein-level? residue-level? both?) needs documenting. Pseudocode + code + 10K-iteration validation.
6. **Trajectory truncation protocol** — all trajectories to T_min per protein (IP §12.1 mandatory). Need: script to identify T_min from Phase 1 data + truncate all Phase 2 trajectories + log per-protein T_min for methods section.
7. **Power analysis recalculation** — IP §1 noted 14×10 design gives ~42% power at ρ=0.5. Now 16 active / 14 S2-counted. Recalculate with ICC correction factor, document on OSF pre-reg.

### Gamma (IP §12.2)
8. **Feature extraction pipeline** — compute RMSF, RSALOR, RSA, ESM2, conservation features from BioEmu batch 1 ensembles (46 proteins already generated). Validate on ProteinGym binding+activity assays (N~57).
9. **Central ablation test** — RSALOR+RMSF vs RSALOR alone, paired Wilcoxon. This is IP Scope Decision #10 (unanimous).
10. **Cross-validation framework** — homolog-aware 5-fold CV with <30% sequence identity threshold. IP §12.2.

### Combined paper (IP §12.3, Appendix A)
11. **JZS Bayesian correlation pipeline** — brms (R) or PyMC (Python) model with Cauchy(0,1) prior. Bridge sampling for Bayes factor computation (Gronau et al. 2017).
12. **4-prior sensitivity analysis** — JZS, Skeptical N(0,0.5²), Informative N(0.5,0.15²), Flat U(-1,1). IP Appendix A.
13. **Crossed random effects model** — (1|protein) + (1|generator); Kenward-Roger + parametric bootstrap (≥1000 iterations).

### Delta (IP §12.4)
14. **WMSE evaluation harness** — hierarchical testing of WMSE → Spearman on top-k DEGs. IP §12.4.
15. **FDR methods implementation** — BH primary, BY sensitivity. IP Scope Decision #4 (unanimous).
16. **Calibration metrics** — reliability diagrams, Expected Calibration Error (ECE), metric gaming tests (random baseline).
17. **Stratified evaluation** — per cell type, perturbation type, expression level.
